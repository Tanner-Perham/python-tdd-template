# Test Driven Development template

### Virtual Environment setup

First ensure that virtualenv is installed:
```bash
pip3 install virtualenv
```

To activate the virtual environment run:
```bash
source venv/bin/activate
```

then to deactivate:
```bash
deactivate
```
